
### `[Instruction Fetch Phase]` — 3 takt
**Takt 1**
- `Memory.addr <- P`
- `IR <- M[P]`
**Takt 2**
- `P <- P + 1`
**Takt 3**
- `Slot_Index <- 0` (переход к декодированию первого слота)

# 2) Arithmetic, Logic and Register Manipulation

## `+` (Plus) — 2 takt
`T <- S + T, Pop data stack`
**Takt 1**
- `ALU <- S + T` *(учитывает Carry, если P9=1)*
- `T <- ALU_out`
**Takt 2**
- `S <- DataStack.pop`
---

## `and` — 2 takt
`T <- S & T, Pop data stack`
**Takt 1**
- `ALU <- S & T`
- `T <- ALU_out`
**Takt 2**
- `S <- DataStack.pop`
---

## `xor` — 2 takt
`T <- S ^ T, Pop data stack`
**Takt 1**
- `ALU <- S ^ T`
- `T <- ALU_out`
**Takt 2**
- `S <- DataStack.pop`
---

## `inv` (Invert) — 1 takt
`T <- ~T`
**Takt 1**
- `ALU <- ~T`
- `T <- ALU_out`
---

## `2*` (Two-Star / Shift Left) — 1 takt
`T <- T << 1`
**Takt 1**
- `ALU <- T << 1`
- `T <- ALU_out`
---

## `2/` (Two-Slash / Shift Right) — 1 takt
`T <- T >> 1` *(знаковый сдвиг)*
**Takt 1**
- `ALU <- T >> 1`
- `T <- ALU_out`
---

## `+*` (Multiply step) — 2 takt
`if A[0]==1 -> T:A = (S+T):A >> 1, else T:A = T:A >> 1`
**Takt 1**
- if `A[0] == 1` -> `ALU <- S + T`
- if `A[0] == 0` -> `ALU <- T`
**Takt 2**
- `T <- ALU_out >> 1` *(сохраняет знак T17)*
- `A <- (A >> 1) | (ALU_out[0] << 17)`
---

# 3) Transfer of Program Control
*(Команды перехода отбрасывают оставшиеся слоты в текущем `IR` и запускают новую фазу `Instruction Fetch`)*

## `name ;` (Jump) — 2 takt
`P <- address`
**Takt 1**
- `P <- IR[destination]` *(адрес берется из остатка слова в IR)*
**Takt 2**
- `Trigger Instruction Fetch`
---

## `name` (Call) — 3 takt
`Push return stack, R <- P, P <- address`
**Takt 1**
- `ReturnStack.push <- R`
**Takt 2**
- `R <- P`
- `P <- IR[destination]`
**Takt 3**
- `Trigger Instruction Fetch`
---

## `;` (Return) — 3 takt
`P <- R, Pop return stack`
**Takt 1**
- `P <- R`
**Takt 2**
- `R <- ReturnStack.pop`
**Takt 3**
- `Trigger Instruction Fetch`
---

## `ex` (Execute) — 3 takt
`Swap P and R`
**Takt 1**
- `temp <- P`
- `P <- R`
**Takt 2**
- `R <- temp`
**Takt 3**
- `Trigger Instruction Fetch`
---

## `unext` (Micronext) — 2 takt
`Loop within IR` *(Уникальная команда: цикл без обращения к памяти)*
**Takt 1**
- `Check R == 0`
**Takt 2**
- if `R == 0` -> `R <- ReturnStack.pop`, перейти к следующему слоту.
- if `R != 0` -> `R <- R - 1`, `Slot_Index <- 0` *(Возврат к 0-му слоту текущего IR)*
---

## `next` (Next) — 2 takt
`Loop to address`
**Takt 1**
- `Check R == 0`
**Takt 2**
- if `R == 0` -> `R <- ReturnStack.pop`, `Trigger Instruction Fetch` *(для следующего слова)*.
- if `R != 0` -> `R <- R - 1`, `P <- IR[destination]`, `Trigger Instruction Fetch`.
---

## `if` — 2 takt
`Jump if T == 0`
**Takt 1**
- `Check T == 0`
**Takt 2**
- if `T == 0` -> `P <- IR[destination]`, `Trigger Instruction Fetch`.
- if `T != 0` -> `Trigger Instruction Fetch` *(P уже увеличен, переход к следующему слову)*.
---

## `-if` (Minus-if) — 2 takt
`Jump if T >= 0 (T17 == 0)`
**Takt 1**
- `Check T[17] == 0` *(проверка знакового бита)*
**Takt 2**
- if `T[17] == 0` -> `P <- IR[destination]`, `Trigger Instruction Fetch`.
- if `T[17] == 1` -> `Trigger Instruction Fetch` *(переход к следующему слову)*.
---
